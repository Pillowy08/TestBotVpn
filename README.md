# 🤖 Telegram VPN Shop Bot

Telegram-бот для магазина VPN с тарифами, реферальной системой и промокодами.

## 📋 Возможности

- **Тарифы VPN** — 4 тарифных плана (1, 3, 6, 12 месяцев)
- **Правила использования** — информация об условиях сервиса
- **Гайд по подключению** — кнопка с инструкцией по настройке
- **Реферальная система** — 10% с каждой покупки приглашённого
- **Промокоды** — система скидок для пользователей
- **База данных** — SQLite для хранения пользователей и покупок
- **Админ-панель** — управление пользователями и промокодами

## 🚀 Быстрый старт (БЕЗ установки Python)

### Вариант 1: Railway (облачный хостинг 24/7) ⭐ РЕКОМЕНДУЕТСЯ

Бот будет работать на удалённом сервере круглосуточно. Ваш компьютер не нужен!

#### Шаг 1: Подготовка

1. **Получите токен бота:**
   - Найдите [@BotFather](https://t.me/BotFather) в Telegram
   - Отправьте `/newbot`
   - Введите имя и username бота (должен заканчиваться на `bot`)
   - Сохраните токен

2. **Узнайте свой Telegram ID:**
   - Найдите [@userinfobot](https://t.me/userinfobot)
   - Отправьте любое сообщение
   - Скопируйте ID

3. **Создайте аккаунт на Railway:**
   - Перейдите на [railway.app](https://railway.app/)
   - Войдите через GitHub (создайте аккаунт на GitHub если нет)

#### Шаг 2: Загрузка проекта на GitHub

1. Создайте новый репозиторий на [GitHub](https://github.com/new)
2. Назовите его, например, `vpn-bot`
3. Сделайте репозиторий **Public** или **Private**
4. Загрузите файлы проекта:

   **Способ A — Через Git:**
   ```bash
   cd "D:\Microsoft VS Code\vpn_bot"
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/ВАШ_USERNAME/vpn-bot.git
   git push -u origin main
   ```

   **Способ B — Через браузер:**
   - Просто перетащите все файлы из папки `vpn_bot` в окно репозитория на GitHub

#### Шаг 3: Развёртывание на Railway

1. Перейдите на [Railway](https://railway.app/)
2. Нажмите **"New Project"**
3. Выберите **"Deploy from GitHub repo"**
4. Найдите ваш репозиторий `vpn-bot`
5. Нажмите **"Deploy Now"**

#### Шаг 4: Настройка переменных окружения

1. В проекте Railway нажмите **"Variables"**
2. Добавьте переменные:
   - `BOT_TOKEN` = ваш токен от BotFather
   - `ADMIN_ID` = ваш Telegram ID

#### Шаг 5: Готово!

Бот запущен и работает 24/7! 🎉

Найдите бота в Telegram и нажмите `/start`

## 📁 Структура проекта

```
vpn_bot/
├── bot.py              # Основной файл бота
├── config.py           # Конфигурация и настройки
├── database.py         # Работа с базой данных
├── keyboards.py        # Клавиатуры и кнопки
├── requirements.txt    # Зависимости Python
├── .env                # Переменные окружения (не в git)
├── .env.example        # Пример переменных
├── .gitignore          # Игнорируемые файлы
└── database.db         # База данных (создается автоматически)
```

## ⚙️ Настройка тарифов

Откройте `config.py` и измените тарифы:

```python
TARIFFS = {
    "1_month": {
        "name": "1 месяц",
        "price": 99,
        "description": "Базовый тариф на 1 месяц"
    },
    # ... другие тарифы
}
```

## 🎁 Создание промокодов

Промокоды можно создавать через админ-панель:

1. Отправьте боту команду `/admin`
2. Нажмите "🎁 Создать промокод"
3. Следуйте инструкциям

Или программно через `database.py`:

```python
from database import add_promo_code
add_promo_code("SALE2026", 50, max_uses=100)  # Скидка 50₽, 100 использований
```

## 🔗 Настройка реферальной системы

В `config.py` измените процент:

```python
REFERRAL_PERCENT = 10  # 10% от покупки реферала
```

## 📖 Настройка гайда

1. Создайте статью на [Telegra.ph](https://telegra.ph) с инструкцией
2. Скопируйте ссылку
3. В `config.py` измените `GUIDE_URL`

## 🛠️ Команды для бота

Настройте команды через @BotFather:

```
start - Запустить бота
tariffs - Показать тарифы
rules - Правила использования
guide - Гайд по подключению
referral - Реферальная система
promo - Активировать промокод
profile - Мой профиль
admin - Админ панель (только для админов)
```

## 📊 База данных

Бот использует SQLite. Таблицы:

- `users` — пользователи
- `purchases` — покупки
- `promo_codes` — промокоды
- `user_promo_codes` — активированные промокоды
- `referrals` — рефералы

## 🔧 Админ-панель

Доступна только пользователям из списка `ADMINS` в `config.py`.

Функции:
- Просмотр всех пользователей
- Статистика бота
- Создание промокодов

## 🐳 Запуск в Docker (опционально)

Создайте `Dockerfile`:

```dockerfile
FROM python:3.11-slim

WORKDIR /app

COPY requirements.txt .
RUN pip install --no-cache-dir -r requirements.txt

COPY . .

CMD ["python", "bot.py"]
```

Запуск:

```bash
docker build -t vpn-bot .
docker run -d --name vpn-bot --env-file .env vpn-bot
```

## ❓ Решение проблем

### Бот не запускается
- Проверьте токен в `.env`
- Убедитесь, что все зависимости установлены
- Проверьте логи на наличие ошибок

### Бот не отвечает на сообщения
- Убедитесь, что бот запущен
- Проверьте, что вы отправили `/start`
- Убедитесь, что токен действителен

### Ошибка базы данных
- Удалите `database.db` и перезапустите бота
- Проверьте права доступа к папке
